## 项目简介
这是一个面向中小型社区团购平台的末端供应链管理系统,提供团长佣金自动核算、订单自动汇总、配送清单一键生成、数据可视化分析等功能。

## 核心功能

### 1. 数据看板
- 今日订单数、销售额、待审核提现、活跃团长等关键指标展示
- 订单趋势折线图(支持近7天/近30天切换)
- 销售额饼图统计
- 团长业绩排行榜

### 2. 订单管理
- 订单列表查询(支持日期范围、状态、团长、关键词筛选)
- 订单详情查看
- 导出配送清单功能
- 导出订单数据功能

### 3. 团长管理
- 团长信息增删改查
- 团长状态启用/禁用
- 佣金比例设置
- 累计订单数和佣金统计
- 账户余额查看

### 4. 佣金管理
- 佣金收益统计(今日/本月/累计)
- 收益趋势图表
- 提现申请列表
- 提现审核功能(通过/拒绝)
- 可提现余额管理

## 业务流程

### 团购流程(7天周期)
1. **第1-3天(预售期)**: 团长推广 → 消费者下单
2. **第4天(截单日)**: 系统汇总 → 生成配送清单
3. **第5天(配送日)**: 供应商配送到团长 → 团长确认收货
4. **第6-7天(自提期)**: 消费者取货 → 团长扫码核销 → 自动计算佣金
5. **月底(结算期)**: 团长查看佣金 → 申请提现 → 管理员审核

### 角色说明
- **平台管理员**: 订单管理、团长管理、数据统计、提现审核
- **团长**: 不负责配送,仅提供自提点、推广商品、核销订单
- **消费者**: 在移动端下单,到团长处自提(本系统不涉及)
