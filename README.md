## 📖 项目简介

本系统是一个面向社区团购业务的管理后台系统(B端),为平台管理员和团长提供订单管理、佣金结算、数据统计等核心功能。

### 核心功能
- 🔐 用户登录与权限管理
- 📊 数据看板与业务统计
- 📦 订单管理与配送清单生成
- 👥 团长管理与业绩统计
- 💰 佣金管理与提现审核

## 🛠️ 技术栈

### 前端技术
- **框架**: Vue 3.x + TypeScript
- **构建工具**: Vite 4.x
- **UI组件库**: Element Plus
- **状态管理**: Pinia
- **路由管理**: Vue Router 4.x
- **数据可视化**: ECharts 6.x

### 后端技术
- **框架**: Node.js 16+ + Express.js
- **数据库**: MySQL 8.0
- **ORM**: Sequelize
- **身份认证**: JWT
- **密码加密**: bcrypt

## 🚀 快速开始

### 环境要求
- Node.js >= 16.0.0
- pnpm >= 8.0.0
- MySQL >= 8.0

### 安装依赖
```bash
# 安装根目录依赖
pnpm install

# 安装前端依赖
cd frontend
pnpm install

# 安装后端依赖(后续添加)
cd backend
pnpm install
```

### 运行项目
```bash
# 仅运行前端
pnpm dev:frontend

# 仅运行后端(后续实现)
pnpm dev:backend

# 同时运行前后端
pnpm dev
```

## 📊 业务流程

```
第1-3天(预售期) → 团长推广 → 消费者下单 → 订单状态:待成团
第4天(截单日)   → 系统汇总 → 生成配送清单 → 订单状态:待配送
第5天(配送日)   → 供应商配送到团长 → 团长确认收货 → 订单状态:待自提
第6-7天(自提期) → 消费者取货 → 团长扫码核销 → 自动计算佣金 → 订单状态:已完成
月底(结算期)    → 团长查看佣金 → 申请提现 → 管理员审核 → 打款
```

## 🎯 核心价值

- ⚡ **效率提升**: 配送清单生成从2-3小时降至<5秒
- 💯 **精准计算**: 订单完成即时计算佣金,减少纠纷
- 🚀 **快速核销**: 核销时长从2-3分钟降至<10秒
